# (In)fra

(In)fra repository collects multiple self made and reusable modules to easily spin up infrastructure for different kind of services; mainly:

- EC2 instances
- Standarized VPCs configurations
- Lambda functions for background jobs
- S3 buckets for static storages
- RDS instances for data storage

The core purpose of this repository is provide an standarized way to create the described infrastructure following the guidelines of the hypotetical **(In)fra** company.
